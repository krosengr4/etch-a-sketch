# Etch-A-Sketch

A browser-based drawing app with a configurable grid, color picker, and erase mode.

- [Quick Start](#quick-start)
- [Features](#features)
- [Usage](#usage)
- [Contributing](#contributing)

## Quick Start

No installation or build step required — just open the file:

```bash
git clone git@github.com:krosengr4/etch-a-sketch.git
cd etch-a-sketch
open index.html
```

On launch, you'll be prompted to choose a grid size (1–100 squares per side).

## Features

- **Configurable grid**: Choose any grid size from 1×1 up to 100×100 on load
- **Color drawing**: Draw with blue, red, yellow, or green
- **Erase mode**: Switch to erase to clear individual squares
- **Reset**: Wipe the canvas and start over without refreshing the page

## Usage

1. Enter a grid size when prompted (e.g. `16` for a 16×16 grid)
2. Hover over squares to draw in the selected color
3. Click a color button to switch colors
4. Click **Erase** to switch to erase mode
5. Click **Reset** to clear the canvas
